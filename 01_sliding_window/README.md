# Sliding Window

Use this folder for all Sliding Window problems.

## When to Use

Sliding Window is useful when the problem asks for something over a contiguous subarray or substring, especially with keywords like:

- longest
- shortest
- maximum/minimum sum
- at most / at least / exactly K
- contiguous subarray
- substring

## Common Patterns

- Fixed-size window
- Variable-size window
- At most K distinct/items
- Exactly K using `atMost(k) - atMost(k - 1)`
- Frequency map inside a window

## Files

- `notes.md` - concepts, formulas, mistakes
- `problems.md` - problem tracker
- problem solution files - one file per problem
